## Color Wheel

[Color picker React component](https://caub.github.io/color-wheel/build)

```js
import ColorWheel from 'color-wheel';

// usage:
<ColorWheel value={color} onChange={setColor} />; // color: [hue, saturation, lightness] (normalized to 0-1)
```

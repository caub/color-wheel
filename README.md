## Color Wheel

[![npm version][npm-image]][npm-url]

```js
import ColorWheel from 'color-wheel';

// usage:
<ColorWheel value={color} onChange={setColor} />;
// color: [hue, saturation, lightness] (normalized to 0-1)
```
